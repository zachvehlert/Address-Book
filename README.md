# README

### Ruby Version:

-  **Ruby**: 3.2.3
### System Dependencies:

- **Ruby**: Make sure Ruby is installed on your system.
- **Rails**: Rails 7.1.3.2
### Database Creation:

1. Create the database:

```
rails db:create
```

2. Migrate the database:

```
rails db:migrate
```

### Run the app:

```
rails s
```

The app will be running on your localhost.
